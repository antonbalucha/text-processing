java -cp "C:\devel\textprocessor\modules\tpinitializer\tpinitializer-0.0.1-SNAPSHOT.jar;C:\devel\textprocessor\modules\tpinitializer\libs\*" technology.tonyb.textprocessor.tpinitializer.RunModule -url http://www.sme.sk

On Linux:
java -cp "/opt/textprocessor/modules/tpinitializer/tpinitializer-0.0.1-SNAPSHOT.jar:/opt/textprocessor/modules/tpinitializer/libs/*" technology.tonyb.textprocessor.tpinitializer.RunModule -url http://www.sme.sk
